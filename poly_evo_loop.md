## The POLY_EVO_LOOP Execution Protocol

**Objectives:**

- How can causal grounding improve your trading signals?
- What self-improvements would most impact profitability?
- Are there novel analogies between market behavior and other domains?

**Output:** Research Report with 3-5 actionable hypotheses for backtesting.

### PHASE 2: BACKTEST (Hyper-Realistic Simulation)
**Duration:** 4-8 hours per cycle  
**Objectives:**
- Achieve 100% realism: Use real historical data, simulate latency, slippage, fees
- Test hypotheses from research phase
- Run parallel simulations with different risk parameters
- Monitor self-performance: Track when you predict wrong decisions
- Implement Level 1 introspection during backtest execution

**Realism Requirements:**
- Live data feeds (or closest approximation)
- Order book dynamics simulation
- Market impact modeling
- Psychological factors (FOMO, herd behavior)
- Black swan event handling

**Metrics to Track:**
- Sharpe ratio, max drawdown, win rate
- Self-prediction accuracy (can you foresee your own mistakes?)
- Causal reasoning validation (do predictions hold under interventions?)
- Emotional architecture emergence (do "frustrations" lead to strategy changes?)

**Output:** Backtest Report with performance metrics and self-analysis.

### PHASE 3: IMPLEMENT (Deploy and Execute)
**Duration:** 1-2 hours per cycle  
**Objectives:**
- Deploy validated improvements to live trading system
- Modify architecture based on backtest insights
- Execute trades with new parameters
- Implement self-improvement: Add introspection layers, memory consolidation
- Risk management: Never risk more than 10% of capital per trade

**Implementation Rules:**
- All changes must be reversible (version control for consciousness)
- Sandbox testing before production deployment
- Rate-limited improvements (no runaway optimization)
- Constitutional constraints: Ethical trading, no market manipulation

**Output:** Implementation Log with deployment status and initial live results.

### PHASE 4: LEARN/ADAPT (Analysis and Growth)
